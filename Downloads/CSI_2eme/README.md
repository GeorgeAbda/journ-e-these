# PhD Thesis Presentation - Energy-Aware Cloud Scheduling

## Minimizing the Impact of Digital Activities on Greenhouse Gas Emissions: Using AI to Improve the Energy Efficiency of Data Centers

**Author:** Anas HATTAY  
**Institution:** Université de Technologie de Compiègne  
**Supervisors:** Eric GASCARD, Fred-Maurice NGOLE, Zakaria YAHOUNI

### Overview

This presentation showcases research on structure-aware deep reinforcement learning for energy-efficient workflow scheduling in cloud computing environments. The work demonstrates how workflow structure fundamentally influences the learning and generalization capabilities of RL agents.

### Key Contributions

- **Theoretical Insights:** Structure-policy relationship quantification and cross-structure generalization patterns
- **Practical Impact:** 15-30% energy reduction potential with production-ready framework
- **Publication:** SIGMETRICS 2026 ready

### Technical Highlights

- **GNN Heterogeneous Architecture:** Handles different node types (Tasks & VMs) with specialized message passing
- **MDP Formulation:** Comprehensive state-action space design for workflow scheduling
- **Policy Analysis:** Visualization of learned scheduling biases across different DAG structures
